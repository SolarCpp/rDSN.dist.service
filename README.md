##### Build

You firstly need to have installed [rDSN](https://github.com/Microsoft/rDSN).

On Linux:
```
git clone https://github.com/imzhenyu/rDSN.dist.service.git xyz
cd xyz
dsn.run.sh build
dsn.run.sh test
```

On Windows:
```
git clone https://github.com/imzhenyu/rDSN.dist.service.git xyz
cd xyz
dsn.run.cmd build Debug .\buildsln
dsn.run.cmd test Debug .\buildsln
```
